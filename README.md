# Project Overview

The AI-Powered Exam Proctoring Camera is an intelligent computer vision system designed to enhance the integrity of offline examinations. The system automatically monitors exam environments to detect suspicious behaviors, including:

- Unusual head movements

- Unauthorized phone usage

By leveraging machine learning and anomaly detection algorithms, the system minimizes false positives while maintaining reliability and accuracy.

This project is aimed at providing a scalable and privacy-conscious solution for exam proctoring, reducing the need for manual supervision while ensuring a fair assessment process.

# Features

- Real-time Monitoring: Detects suspicious behaviors live during exams.

- Anomaly Detection: Algorithms identify irregular movements or gestures indicative of potential cheating.

- Phone Usage Detection: Monitors for unauthorized phone presence or usage.

- Privacy-Conscious: Logs and alerts suspicious behavior without storing unnecessary personal data.

- Collaborative Development: Developed in a team environment with code reviews, shared responsibilities, and iterative improvements.
